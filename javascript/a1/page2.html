In the first semester, I developed a diverse set of skills that are highly valued in the technology industry. These skills are essential for a career in technology or computer science.

HTML/CSS: I developed a solid understanding of how to create and structure web pages using HTML and how to style them using CSS. These skills are essential for creating and designing websites and web applications. I are now able to create responsive, mobile-friendly and visually appealing web pages.

Python: I learned how to use Python as a programming language, which is widely used in the industry for various tasks such as data analysis, machine learning, and web development. With my Python skills, I are now able to write scripts and programs that can automate repetitive tasks, extract data from websites, and perform complex calculations.

Networking: I developed knowledge and understanding of how networks work, how data is transmitted and how to troubleshoot network issues. I learned about protocols, topologies, and how to secure networks. This skill will be useful in various career paths such as Network administrator, Security analyst, and IT support.

Linux: I have developed the skills to use Linux as an operating system, which is widely used in the industry. I are now able to navigate the command-line interface, manage files and directories, install software and troubleshoot issues. These skills will be useful in various career paths such as System administrator, DevOps engineer, and Network administrator.

Communication: I have developed my communication skills by working in group projects, and by presenting my work in front of my peers. This skill will be useful in any career path, as it is essential to be able to effectively communicate my ideas, solutions and findings to my team, customers and stakeholders.

Overall, the skills I developed during the first semester will serve as a foundation for my future studies, and open many doors to different career paths in the technology industry. I have a solid foundation in web development, programming, networking, and Linux, which are in high demand in the job market. And with the added bonus of good communication skills, I are well-positioned to succeed in any role that I choose to pursue.
